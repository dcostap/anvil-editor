# Symbol-kind icons

Most SVGs in this directory come from JetBrains' Apache-2.0 IntelliJ Platform
Experimental UI icon set (`platform/icons/src/expui/nodes`). The copyright and
license notice is retained in each imported SVG; see `LICENSE.txt` for the full
license.

`struct.svg`, `struct_dark.svg`, `union.svg`, and `union_dark.svg` are
Anvil-authored additions using the same Apache-2.0 visual style.
